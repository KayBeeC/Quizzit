About

Quizzit is a web application which delivers quizzes to people who want to challenge themselves on specific topics and categories against the world.
The website is easy to navigate through, you can easily create an account using the Sign Up button or simply play quizzes as a guest.

Account Privileges

All types of users will be able to play multiple choice quizzes from all genres in the easy to medium difficulty categories. Registered users will be able to play the hard difficulty and will be able to access more features in the application. Users will be able to view a leader-board, (only listing registered users) where they can see if their attempt was excellent, adequate or worse in comparison to other users. This will be based off the number of correct answers and time remaining for each quiz the user attempts. This adds a competitive edge making it an increasingly challenging and fun concept of doing quizzes.
